使用说明

  1. 打开 Google Scholar
  2. 搜索并进入您的个人主页
  3. 复制地址栏中的完整 URL
  4. 粘贴到上方的输入框中
  5. 点击"保存设置"

部署自己的代理（可选）

为了更稳定的访问，您可以部署自己的 Cloudflare Worker：

  1. 访问 Cloudflare Workers
  2. 创建新的 Worker
  3. 复制 cloudflare/worker.js 中的代码
  4. 部署并获取 Worker URL
  5. 在上方填入 Worker URL
